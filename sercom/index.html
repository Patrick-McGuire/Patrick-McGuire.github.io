SAMD21 / SAMD51 SERCOM Configurator

Plan SERCOM assignments (I²C / SPI / Serial) for custom boards. Data verified against the Microchip SAM D21 (Atmel-42181N) and SAM D5x/E5x (DS60001507F) datasheets and the Adafruit ArduinoCore-samd SERCOM library.

Configure
Summary & Arduino code

Pick a protocol per SERCOM, then a pin per signal. Every option stays visible so you can experiment. In the dropdowns: green = your current valid pick; red = a problem — either the pin is already used by another SERCOM signal (in use) or it's incompatible with your other picks here (wrong pad layout, or a different IOSET on SAMD51). Red options are still selectable — problems are flagged, not blocked. A pin marked ALT is PIO_SERCOM_ALT; unmarked pins are plain PIO_SERCOM. When only one pin set is valid it is pre-selected.
