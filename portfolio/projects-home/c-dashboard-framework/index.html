C++ Dashboard Framework

I wrote a dashboard framework for displaying real time data. I had worked on python versions of this project previously, but I wanted to learn C++ and write code with a structure that was well thought out. Supported features include user input event handling, multi-threaded data input streams, real-time graphing and a flexible objected oriented codebase. I used Doxygen to maintain documentation. 

Although it's still a work in progress, I learned a lot about memory allocation, multithreading, design patterns, and code documentation through work on this project.

You can view the codebase and documentation here:
