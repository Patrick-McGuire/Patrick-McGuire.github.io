Apeiron Labs:
Hydraulic Pump Characterization System

Overview

I developed a system to test hydraulic pumps automatically. The system was controlled by a Arduino, and could set pump motor's RPM and the induced pressure. It could measure oil temperature, motor temperature, true pressure, flow rate and power consumption. The Arduino could be configured to run various procedures and record the data automatically. I developed documentation including safety procedures, a user guide, and a data analysis pipeline for processing the data. I used the system to assist in pump selection for the robot Apeiron Labs is currently developing, and the characterized and verified all pumps used in production. I worked with the software developers to make controls decisions based on the data the system recorded.
