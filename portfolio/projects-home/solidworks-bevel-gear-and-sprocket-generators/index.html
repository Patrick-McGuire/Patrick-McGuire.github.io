SolidWorks: Bevel Gear and Sprocket Generators

Overview

In order to learn more about gear design, I created parametric bevel gear and sprocket generators for SolidWorks. Build entirely within part files using the equation system, these are very easy to use. One part file will generate a sprocket based on trade number and teeth. The second file will generate a pair of bevel gears based on gear pitch, numbers of teeth, shaft angle, pressure angle, and other tunable parameters. The bevel gear generator uses true involute curves.

These files are publicly available, and have been downloaded over 1000 times: https://grabcad.com/library/sprocket-and-bevel-gear-generators-1
