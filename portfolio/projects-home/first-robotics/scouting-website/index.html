Competitive Robotics Scouting Website

I created a website for tracking opponent teams throughout a robotics competition. The website consisted of forms for entering data, and various data visualizations. The system was written with google apps scripts/JavaScript, and used google sheets for data storage and detailed analysis. The website would generate graphs and tables for basic data review. A earlier version of this system was first used in 2019, and is still used by my high school team.
