Engineering Consulting:
Aquaculture Automatic Oyster Sorting

Thermal camera system overview

Overview

In 2020 I worked as a consultant for an oyster-farming company, where I developed software to track oyster temperature and size to help customers receive consistently fresh oysters. The system automatically removed oysters that were too small using pneumatics, then collected data on the remaining large oysters using a combined thermal and video camera.  I wrote software to extract temperature based on this data, then uploaded images and data to to Amazon Web Services. I used OpenCV Python to detect and track motion on the conveyor belt.
